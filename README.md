# La Roue Gourmande - Guide de Déploiement

Ce projet est prêt à être hébergé sur GitHub Pages.

## Comment mettre en ligne sur GitHub

1.  **Créer un compte GitHub** (si ce n'est pas déjà fait) sur [github.com](https://github.com).
2.  **Créer un nouveau dépôt (Repository)** :
    *   Cliquez sur le bouton "+" en haut à droite -> "New repository".
    *   Nommez-le (ex: `roue-boulangerie`).
    *   Cochez "Public".
    *   Cliquez sur "Create repository".
3.  **Uploader les fichiers** :
    *   Dans la page de votre nouveau dépôt, cliquez sur "uploading an existing file".
    *   Glissez-déposez les fichiers `index.html`, `style.css`, et `script.js` que j'ai créés.
    *   Cliquez sur "Commit changes".
4.  **Activer GitHub Pages** :
    *   Allez dans l'onglet "Settings" du dépôt.
    *   Dans le menu de gauche, cliquez sur "Pages".
    *   Sous "Build and deployment" > "Source", choisissez "Deploy from a branch".
    *   Sous "Branch", sélectionnez `main` (ou `master`) et le dossier `/ (root)`.
    *   Cliquez sur "Save".
5.  **C'est en ligne !**
    *   Attendez quelques minutes. GitHub vous donnera un lien (ex: `https://votre-pseudo.github.io/roue-boulangerie/`).
    *   C'est ce lien que vous pourrez partager ou afficher sur une tablette dans la boulangerie.

## Tester le jeu

*   **Horaires** : Le jeu ne fonctionne qu'entre 7h et 20h. Si vous testez en dehors de ces heures, changez l'heure de votre appareil.
*   **Une fois par jour** : Le jeu empêche de rejouer le même jour (via les cookies du navigateur). Pour rejouer, ouvrez le site en "Navigation Privée" ou videz le cache.
